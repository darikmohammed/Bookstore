# Book Store

> "Book Store" is a react application which have a basic functionality of

- Display a list of books.
- Add a book.
- Remove a selected book.

## Built With

- React
- Jest-test
- Redux

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Visual Studio code or any Code Editor.

### Setup

#### Download the file to local directory

- if you have already insall github in your local pc run:

```sh
git clone git@github.com:darikmohammed/Bookstore.git
```

- Or download the zip file from the github directory.

### Run the project locally

```sh
npm install
```

```sh
npm start
```

#### Linters tests

- Eslint
- Stylelint
- node_modules checker
